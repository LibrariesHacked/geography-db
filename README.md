# Geography database

A database to hold common UK geography.

## Datasets

- [Countries (December 2022) UK BFC](https://geoportal.statistics.gov.uk/datasets/ons::countries-december-2022-uk-bfc/explore?location=55.215430%2C-3.316939%2C6.52)
